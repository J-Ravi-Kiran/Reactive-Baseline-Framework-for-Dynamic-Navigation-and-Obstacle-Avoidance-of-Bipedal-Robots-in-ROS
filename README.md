# Reactive-Baseline-Framework-for-Dynamic-Navigation-and-Obstacle-Avoidance-of-Bipedal-Robots-in-ROS
This project presents a lightweight reactive baseline framework for biped robots developed in ROS Noetic and Gazebo simulation. A custom URDF-based biped robot model and a sinusoidally moving obstacle were used for evaluation. Twenty experimental trials were conducted under identical conditions.
