# Reactive-Baseline-Framework-for-Dynamic-Navigation-and-Obstacle-Avoidance-of-Bipedal-Robots-in-Simulated-Indoor-Environments
This project presents a lightweight reactive baseline framework for biped robots developed in ROS Noetic and Gazebo simulation. A custom URDF-based biped robot model and a sinusoidally moving obstacle were used for evaluation. 

 The proposed framework, inspired by existing reactive avoidance techniques for bipedal platforms, integrates a finite-state machine (FSM) navigation controller with a simple lateral sidestepping strategy for dynamic obstacle avoidance, enabling continuous goal-directed motion without complex perception or prediction modules
